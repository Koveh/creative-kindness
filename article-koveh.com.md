# Website with blog engine and payment system for charity organization.


Design-first approach.


The sidebar is an point of attention




Сайт для журнала о благотворительности.

Сайт с блогом, системой оплаты и адиинкой. Написан на React Next.js. Отличительная черта – сайдбар – навигация слева.

Навигация не мешает читать статьи, которые находятся в центре. 

Текст расширяется до ширины, при которой удобно читать по диагонали.  

Мобильная версия позвляет читать лонгриды. используется текст без засечек для удобства чтения с малых экраоноав. Сдержанный Suiss intl показывает принадлежность к модному сообществу gen x и gen z, которые ценят дизайн, как определение качества и доверие к сервису.

Сначала создали варианты дизайна сайта в Figma.

сверстали сайт в React Next.js без дизайна, только с компонентами. Протестировали удобность на мобильных устройствах и ноутбуках.

После - добавили базу данных на PostgreSQL и S3 на Minio (бесплатный аналог AWS S3).

сделали админку на React Next.js и создали движок для блога: черновики, кэширование (локальное сохранение менений), транслит ссылок, сохранение фото и сжатие в формат webp и хранение в S3.

Сделали систему оплаты на Stripe для теста. потом перевели на Yoomoney.
